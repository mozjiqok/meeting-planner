# Бот для записи на созвоны (Telegram Meeting Scheduler) 🔥

Этот проект представляет собой Telegram-бот для автоматизации записи подписчиков на консультации или созвоны. Бот ограничивает количество звонков в день и вводит дополнительные вопросы для фильтрации нецелевых запросов.

## 🚀 Основные возможности

- **Telegram бот** (Laravel + Nutgram):
  - Проверка подписки на канал перед началом диалога.
  - Опросник из 3-х шагов «для фильтрации» (полностью на русском).
  - Умный выбор слотов (Пн-Пт 07:30 и 17:30) на 14 дней вперед.
  - Защита от одновременного бронирования одного и того же времени.
  - Напоминания за 24 часа и за 1 час через системный планировщик.

- **Админка (Web + Telegram)**:
  - **Web**: Управление слотами, просмотр ответов подписчиков, установка ссылок на звонок и ручная отмена встреч.
  - **Telegram**: Быстрые команды `/slots`, `/bookings`, `/block` для администратора.

- **Infrastructure**:
  - Полная поддержка **Docker** (Nginx, PHP-FPM, Queue Worker, Scheduler).
  - SQLite по умолчанию для максимально простого деплоя.

## 🛠 Инструкция по запуску

### 1. Подготовка окружения

Клонируйте проект и выполните команду настройки:

```bash
make setup
```

Это создаст файл `.env`, базу данных SQLite и сгенерирует ключи приложения.

### 2. Настройка конфигурации

Откройте `.env` и укажите данные вашего бота:

```env
APP_TIMEZONE=Asia/Vladivostok

# Токен из @BotFather
TELEGRAM_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11

# Ваш личный ID (узнайте через @userinfobot)
TELEGRAM_ADMIN_ID=123456789

# Юзернейм канала для проверки подписки
TELEGRAM_CHANNEL_USERNAME=@mychannel

# Доступы в Web-админку (по умолчанию http://localhost/admin)
ADMIN_EMAIL=admin@local
ADMIN_PASSWORD=your_secure_password
```

> [!TIP]
> Чтобы изменения пароля админа вступили в силу после правок в .env, запустите:
> `php artisan db:seed --class=AdminUserSeeder`

### 3. Запуск через Docker

```bash
make up
```

Команда поднимет все необходимые сервисы. Веб-интерфейс будет доступен на порту 80.

### 4. Регистрация вебхука (для Production)

Если сервер доступен извне по https:

```bash
make webhook URL=https://your-domain.com
```

## 👨‍💻 Команды управления (Makefile)

- `make up` / `make down` — запуск и остановка Docker.
- `make logs` — просмотр логов всех сервисов.
- `make seed` — сброс и заполнение слотов по умолчанию.
- `make webhook-info` — проверка статуса вебхука в Telegram.
- `make test-reminders` — ручной запуск проверки и отправки напоминаний.
